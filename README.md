## ✦ About Me

I’ve been fascinated by computers for as long as I can remember. What drew me to software engineering was not simply learning how technology works, but gaining the freedom to transform an idea into something real.

I enjoy the complete engineering journey  identifying a problem, planning thoughtful features, designing the user experience, developing and testing the solution, and considering how it can remain secure and trustworthy.

My interests lie at the intersection of **software engineering and cybersecurity**. I hope to build original, user-centered systems while continuing to explore application security, secure software development, and the way systems can be both protected and responsibly tested.

---

## ✦ My Engineering Philosophy

> Great software should not only work  it should solve a meaningful problem, respect its users, and be secure by design.

I begin projects by understanding the problem before choosing the technology. I value creativity, ethical decision making, thoughtful design, and quality throughout the software development lifecycle.

## ✦ Things I’ve Built

### 🚗 Basirah | AI Smart Parking & Navigation System

A smart parking solution developed during a 48-hour MemryX Digital Hackathon to help Prince Sultan University users find, reserve, and navigate to available parking spaces.

**My contribution**

- Proposed the project concept after identifying recurring parking challenges at the university
- Helped define core features including identity verification, real-time availability, reservations, in-app navigation, notifications, and parking verification
- Contributed to the application’s visual identity, color palette, user flows, and Flutter interface
- Helped take the project from initial concept to final presentation during the 48-hour hackathon
- Prepared the project presentation and helped communicate the final concept to the judges

---

### 🧠 Soluna — Mental Wellness Software Testing Project

A privacy-focused mental wellness platform designed around journaling, mood tracking, community interaction, and user privacy.

**My contribution**

- Contributed to the test plan, test specifications, and structured test cases
- Applied black-box techniques including equivalence partitioning and boundary value analysis
- Worked on unit and integration testing scenarios for application workflows
- Considered privacy, authentication, access control, and secure data handling throughout the testing process

---

### ✈️ WanderLog — Travel Scrapbook Platform [wanderLog](https://github.com/Aster-L9/wanderlog)

A travel-focused web platform combining destination planning, personal journals, interactive experiences, and a subscription-based service model.

**My contribution**

- Designed and implemented the Home and Payment interfaces using HTML and CSS
- Developed responsive layouts for different screen sizes
- Created a consistent visual system with light and dark themes
- Refined typography, spacing, pricing components, and interface consistency across the assigned pages
- Collaborated with the team through an Agile development process

---

### 🔐 OpenClinica — Secure Software Engineering Assessment [OpenClinica](https://github.com/Aster-L9/OpenClinica-)

A security assessment of a clinical research platform focused on protecting sensitive patient and research information.

**My contribution**

- Applied STRIDE threat modeling to identify threats affecting critical system assets
- Traced assets to architectural components and relevant security risks
- Used SQUARE-Lite principles to develop security requirements
- Defined controls involving authentication, RBAC, MFA, audit logging, account protection, and encryption
- Evaluated risks affecting confidentiality, integrity, and availability

---

### 🛡️ SFDA — Security Risk Management Case Study

A cybersecurity assessment examining the security posture of an e-government platform and its protection of sensitive information.

**My contribution**

- Examined vulnerabilities, authentication practices, confidentiality controls, and organizational security risks
- Evaluated risk likelihood and impact to support risk prioritization
- Proposed improvements involving MFA, RBAC, encryption, intrusion detection, patch management, and security awareness
- Assessed how the recommended controls strengthen confidentiality, integrity, and availability
